Help Topics

Adding a new Journal :

To add a new journal, click the Journal Icon and complete the form - Title, Time and Tags are mandatory fields

Deleting a Journal :

Click the delete link to remove the associated journal. The page will refresh excluding the deleted journal

Editing a Journal :

To edit a journal click the edit link and amend the journal. Clicking the cancel button or the journals link will cancel any edits and bring you back to the all journals view

Time :

Time is a mandatory field and indicates the time spent on the journal. This is in freefoirmn text

Learnings :

Learnings is not manadatory. This indicates what learning were achieved on the journal entry

Resources :

Field indicates what resources were used

Tags :

Muttiple tags can be entered split by a ',' e.g. 'Holiday, Scotland'. Without the comma the entry will seen as a single tag.


A tag is also clickable. Once clicked all journals with the same tag will be shown
